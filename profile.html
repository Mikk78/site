What is 2FA?

2FA is a method for determining the authenticity of a user, which requires passing two degrees of protection (and not one, as before) before gaining access to an account.

    These security levels are:
  • account password;
  • a unique digital code that is generated by a special application installed on a mobile device.

What is Google Authenticator? Google Authenticator is a 2FA application that generates a unique 6-digit code every 30 seconds. There are applications for Android and iOS.

This can happen because time is running out. To fix this, go to the Google Authenticator application and click on the button in the upper right corner to open the menu in which select “Settings”. Then “Time Correction for Codes”, then click the “Synchronize” button.
In most cases it helps. If this did not work, then the only option is to write in support so that they deactivate 2FA for your account and you could log in using only the password. But be prepared for the fact that in return you may be asked for different information about your account - what kind of balance, what kind of coins are there, what operations have you performed lately, what did you buy, what did you sell, etc.
The next problem that anyone can face is when something happens to your mobile device. It can be stolen, it can break, you can accidentally delete both the Google Authenticator application itself and any account in it.

We will understand for a start with the last item. You can have one account in the application, or hundreds of different ones. And in order not to delete the necessary one by mistake, we recommend not to leave the default names, but rename the accounts for services. Then it will be clear what kind of account.
In case of theft or breakdown of the device, always save the QR codes as a picture, and 16-digit keys in text form or copy to paper. If your code and key are stored in a safe place, then you can always access your account.
